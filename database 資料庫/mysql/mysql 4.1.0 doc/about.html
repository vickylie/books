希望一個月的努力能為 MySQL 的推廣盡一份綿薄之力。

一直以來都使用晏幾 (clyan@sohu.com) 翻譯的 MySQL 3.23.7 alpla ，但是隨著 MySQL 版本的不斷發展，越來越發現這一版本以不能適應 MySQL 發展了，為了能更多的人更清楚地了解與理解 MySQL 所提供的新函數，新功能，新表類型等等，就一直想做這麼一件事。

由於本人水平有限，文中肯定有翻譯不準確的地方，敬請在閱讀過程中不吝指出。文中不明之處，請參閱手冊原文 MySQL Reference Manual 。

本文內容針對於 MySQL 4.1.0 - alpha。

MySQL官方站台：http://www.mysql.com
